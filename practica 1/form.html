ETIQUETAS PARA FORMULARIOS

<form></form>

El elemento HTML <form> Esta etiqueta no cambia ni nos da un estilo en particular, tampoco modifica el contenido de la pagina,sin embargo si representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web.
Es posible usar las pseudo-clasess de CSS :valid e :invalid para darle estilos a un elemento form.

<input></input>

El elemento HTML <input> Eqtiqueta para la entrada de datos se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario.
El elemento <input> es uno de los más potentes y complejos en todo HTML debido a la gran cantidad de combinaciones de tipos y atributos de entrada. Algunos atributos del <input> son:

      button
      checkbox
      date
      datetime-local
      email
      file
      number
      password
      radio
      submit
Ejemplos:
  1. <label for="password">Ingrese su contraseña</label>
    <input type="tipo de input" name="contraseña" id="password"></input>
  2. <input type="submit" value="Confirmar"></input>
    <button type="submit">Enviar</button>
  3. <label for="pais-1">seleccione un pais</label>
    <span>Argentina</span><input type="radio" name="pais-1" id="Ar"></input>
    <span>Francia</span><input type="radio" name="pais-1" id="Fr"></input>
    Argentina Francia
  4. <label for="mensaje">Ingrese su mensaje</label>
    <textarea name="mensaje" id="mensaje" cols="30" rows="10" placeholder="hola a todos"></textarea>

<label></label>

El Elemento HTML <label> representa una etiqueta para un elemento en una interfaz de usuario.Este puede estar asociado con un control ya sea mediante la utilizacion del atributo for, o ubicando el control dentro del elemento label.

<label for="lo mismo que el id">Nos sirve para saber de que trata nuestro input</label>
<input type="tipo de input" name="nombre de usuario" id"lo mismo que el name" placeholder="indica que tipo de dato debemos ingresar"></input>

atributos: for

El ID del elemento de formulario etiquetable relacionado en el mismo documento que el elemento label. El primer elemento en el documento con tal ID que coincida con el dispuesto en el atributo for será el control etiquetado para este elemento.

<label for="email">Ingrese su Email</label>
<input type="email" name="Email" id="Email"></input>

<select></select>

Este elemento de HTML representa un control que muestra un menú de opciones. Las opciones contenidas en el menú son representadas por elementos <option></option>.

<label for="pais">selecciona un pais</label>
<select name="pais" id="pais">
<option>argentina</option>
<option>peru</option>
<option>uruguay</option> </select>
